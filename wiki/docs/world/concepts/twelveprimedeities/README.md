# 十二主神的由来
十二主神并不是严格遵守十二这个数字，实际上，并没有完全遵守十二这个数字的神话体系，十二主神，只是被某位神学家整理出来的，在各个神话体系中出现频率最高的神职，虽然谁也没见过神明的诞生，但是据他研究，神明的诞生于这些神职之中，至于是被赋予，还是经由争夺，这就不得而知了。
有的神系将神职聚于一身，成为唯一神系；有的神系则贴近十二主神的分工，平衡而和谐；而有的神系，不断地将神职分裂，最终塑造了万物皆神的体系。这些神系虽然各有各的架构，但是也各有各的强，并不能粗浅的将其分个强弱。
